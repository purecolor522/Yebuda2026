## 📱 第一輪：手機 Safari（真機，最重要）

### A. 視覺 / RWD
- [ ] 首頁打開**沒有左右捲動條**（這是最關鍵的）
- [ ] Top bar 通告「全館新品上架」一行能讀，不會被截斷
- [ ] Logo「Yebuda 漂亮小姐」清晰可讀，沒擠到右邊 icons
- [ ] Hero 大圖佔據合理高度（不會太矮、不會撐爆螢幕）
- [ ] 商品區塊是 **2 欄**（不是 1 欄也不是 3 欄）
- [ ] 「上衣 / 下身 / 洋裝 / 外套 / 套裝 / 特價」分類 tab 可以**橫向滑動**
- [ ] Lookbook 圖片標題直接顯示（不需 hover）
- [ ] Footer 4 個區塊變成 1 欄堆疊
- [ ] LINE / IG / Email 三個圖示間距夠，能用手指準確點到

### B. 主要購物流程
- [ ] 點擊任何商品 → 彈出商品 modal，圖片完整顯示
- [ ] 尺寸按鈕（S/M/L）能輕鬆點選
- [ ] 數量 +/- 按鈕能準確點
- [ ] 「加入購物車」→ 購物車抽屜從右邊滑入，**佔滿螢幕**
- [ ] 抽屜可以上下捲動看商品
- [ ] 「前往結帳」→ 表單能完整填寫（不會被鍵盤擋住關鍵欄位）

### C. 會員功能
- [ ] 點擊 👤 → 登入彈窗能完整顯示，按鈕不被切掉
- [ ] 註冊填完表單能成功送出
- [ ] 登入後再點 👤 → 會員面板可看到「我的基本資料」
- [ ] 「編輯」→ 表單可填、可儲存
- [ ] 「📦 我的訂單」→ 儀表板**全螢幕模式**、可以滾動看訂單

### D. 連結 / 跳轉（最容易漏的）
- [ ] Footer 點「上衣」→ 自動切到上衣分類 + 滾到 #products
- [ ] Footer 點「人氣排行」→ 滾到 Best Sellers 區塊
- [ ] Footer 點「常見問題」→ 彈出 5 題 Q&A
- [ ] Footer 點「聯絡我們」→ 彈出 LINE / IG / Email 三個聯絡方式
- [ ] 點 **IG 圖示** → 開 Instagram app（或網頁）
- [ ] 點 **LINE 圖示** → 跳到 LINE app 加好友（這個一定要實機測試）
- [ ] 點 **Email 圖示** → 跳出郵件 app 預設收件人 yebuda22@gmail.com
- [ ] 點 ↑ 回到頂部按鈕能正常作用

### E. 漢堡選單
- [ ] 左上角 ☰ 點下去能展開選單
- [ ] ✕ 能關閉選單
- [ ] 選單內「登入/註冊」→ 開登入彈窗
- [ ] 選單內「訂單查詢」→ 已登入直接開儀表板，未登入提示先登入

## 💻 第二輪：桌機 Chrome DevTools 模擬

開 `Cmd + Option + I` → `Cmd + Shift + M`，跑下列裝置：

- [ ] **iPhone SE (375×667)** — 最小現役 iPhone，最容易出問題
- [ ] **iPhone 14 Pro (393×852)** — 主流尺寸
- [ ] **iPad Mini (768×1024)** — 平板分隔線（768px 是 RWD 斷點）
- [ ] **iPad Pro 12.9 (1024×1366)** — 大平板
- [ ] **桌機 1280px** — 設計的主要寬度

每個尺寸都要：
- [ ] 沒有水平捲動條
- [ ] 文字不會疊在一起
- [ ] 按鈕不會被切到只剩一半

## 🎯 第四輪：完整下單流程（end-to-end）

用真的 iPhone 走一次：

1. [ ] 註冊新帳號（用真實 email）
2. [ ] 加 2-3 件商品到購物車
3. [ ] 結帳填地址、選付款方式 → 送出
4. [ ] 收到訂單成立通知（信箱 / 後台）
5. [ ] 後台用 Mac 把訂單標為「📦 出貨」
6. [ ] 回 iPhone →「我的訂單」應該看到「已出貨」+ 物流單號
7. [ ] 點「退貨」→ 後台應該收到退貨申請

跑完這 7 步沒卡住 = 整個系統真的能上線。

## 📝 怎麼用真機測

```bash
# 找 Mac IP
ipconfig getifaddr en0
# 假設輸出 192.168.1.23

# 啟動 dev server
npm run dev
```

iPhone 跟 Mac 連同一個 Wi-Fi → Safari 打開 `http://192.168.1.23:3000`

---

## 🔍 怎麼看 iPhone Safari 的 console error

1. iPhone：設定 → Safari → 進階 → 打開「JavaScript」+「網頁檢閱器」
2. iPhone 用 USB 連 Mac
3. Mac Safari → 開發 選單 → 找到你的 iPhone → 點開分頁
4. 即可在 Mac 上看 iPhone 上的 console、network、DOM
